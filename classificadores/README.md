# Instruções para execução

##1) Instalação

```
 $ unzip classificadores.zip
 $ cd classificadores
 $ virtualenv venv
 $ source venv/bin/activate
 $ pip install -r requirements.txt
```

##2) Execução

```
 $ python classificadores.py <treino> <teste>
```
